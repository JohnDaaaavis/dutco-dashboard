📤 How to Share This Portal

Free hosting on GitHub Pages (5 mins setup)

1
Go to github.com and create a free account (or log in)
2
Click + New repository — name it dutco-dashboard, set to Public
3
Click uploading an existing file → drag in this HTML file → rename it to index.html → click Commit changes
4
Go to Settings → Pages → under Source select Deploy from a branch → main → / (root) → Save
5
Wait ~60 seconds. Your permanent link will be:
https://YOUR-USERNAME.github.io/dutco-dashboard

Every week — update the report

  1. Open the link on any device (phone, laptop, tablet)
  2. Drop in the new Excel files for that week
  3. The report generates instantly — no reload needed
  4. Share the same permanent link via Email / Teams / WhatsApp
🔒 Privacy note: Excel files are read entirely within the browser. Nothing is uploaded to any server. GitHub only hosts the HTML file itself — your data never leaves the device of whoever is viewing the report.

Alternative: OneDrive / SharePoint

Save this HTML file to a SharePoint document library → right-click → Copy link → share. Anyone with access can open it and drop in their files directly.
